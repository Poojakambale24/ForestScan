About this Project

This dashboard visualizes tree-level change detection results derived from satellite imagery.

  • NDVI trends per Taluka
  • GLCM texture features to show structural changes
  • Automated zone classification by NDVI slope

Zone rules: Red (slope < -0.02), Orange (-0.02 ≤ slope < 0.01), Green (slope ≥ 0.01)
